ABSTRACT
================================================================
[KEEP] The Health & Meal Management System project is a mobile application developed using
the Flutter framework and Firebase cloud services. The objective of this project is to design
and implement a user-friendly application that helps individuals track their daily meals,
manage dietary habits, and maintain healthier lifestyles.

[KEEP] The application allows users to record, update, and monitor their meal information
while securely storing the data using Firebase Cloud Firestore.

[KEEP] Core functionalities: user authentication, meal logging, meal editing, and real-time
data synchronization via Firebase Authentication.

collaboration (create groups, send invitations, manage group participation).
